About

Toolbox of automated tests for web interfaces (mainly forms) capable of testing navigation, interaction with the client, and use cases.

How to use

@annotation arg1 arg2 ...

Annotation

Annotation Parametres Description
@open url (form url) open the browser
@close none close the browser
@click id It allows you to click on an element in the form by providing its ID.
@InputFormWrite id / value send a value to the form. takes an id and fills a value to any element of the form. It fills all types of input (file, radio, ...)
@InputFormRead id read a field and do something with the results
@compareText id / sqlRowTobeCompare / sqlRequete compare an innertext with an SQL query
@compareInnerHTML id compare the HTML code of the id with an HTML code
